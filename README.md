# Fashion Store - Mini E-Commerce Website

A simple, beginner-friendly e-commerce clothing store website with a clean and professional design.

## Features

- **4 Pages**: Home, Products, Product Detail, and Shopping Cart
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Professional color scheme with good contrast
- **Shopping Cart**: Add products to cart with size and quantity selection
- **Local Storage**: Cart data persists using browser's localStorage
- **Minimal JavaScript**: Easy to understand and modify

## Pages

1. **Home Page (index.html)**: Hero section with featured products
2. **Products Page (products.html)**: Display all available products
3. **Product Detail Page (product-detail.html)**: View product details, select size and quantity
4. **Cart Page (cart.html)**: View cart items and proceed to checkout

## How to Use

1. Open `index.html` in your web browser
2. Navigate through the pages using the navigation menu
3. Click on any product to view details
4. Add products to cart with your preferred size and quantity
5. View cart and proceed to checkout

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- Vanilla JavaScript (ES6)
- LocalStorage API

## Browser Support

Works on all modern browsers:
- Chrome
- Firefox
- Safari
- Edge

## Customization

You can easily customize:
- Colors: Edit CSS variables in `style.css`
- Products: Modify the products object in `js/product-detail.js`
- Images: Replace placeholder images with your own product images

## Notes

- This is a front-end only project (no backend)
- Cart data is stored in browser's localStorage
- Product images use placeholder URLs (replace with your own images)
